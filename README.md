# HomeCue
HomeCue is a simple Python service that runs alongside Corsair iCue.  It exposes your Corsair RGB lighting hardware and makes them auto-discoverable MQTT devices which can be bridged between iCue and Home Assistant.

 - Configure by running the setup script in powershell.
 - Uses MQTT broker through HA to bridge with iCue.
 - See DOCUMENTATION.md for full setup instructions and features.
